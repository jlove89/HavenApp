‚ùì Troubleshooting

‚ùå I see a blank page or error

  1. Try refreshing the page: Press F5
  2. Wait 10 more seconds
  3. Check that both terminal windows are still running
  4. Try again: http://localhost:8080

‚ùå It says "Python not found"

  1. Download Python from: https://python.org
  2. Run the installer
  3. IMPORTANT: Check the box "Add Python to PATH"
  4. Restart your terminal
  5. Try launching again

‚ùå It says "Flutter not found"

  1. Download Flutter from: https://flutter.dev
  2. Follow the installation guide for your operating system
  3. Restart your terminal
  4. Try launching again

‚ùå Port 5000 is already in use

Something else is using that port.

Quick fix: Restart your computer and try again.
